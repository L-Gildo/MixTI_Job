Um pager simples construído em torno de um código HTML / CSS / JavaScript (alimentado pela grandiosidade do SVG).
Deve funcionar muito bem para uma página de destino. Também inclui estilo padrão para vários
elementos de página padrão (Obs.: há varias linhas de código comentadas para não encher apágina que aprincípio será mais enxuta).

Imagens de demonstração* cortesia de Unsplash, uma coleção incrível de imagens CC0 (domínio público)
você pode usar para praticamente qualquer coisa.
